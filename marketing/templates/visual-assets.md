# Visual Asset Guidelines

## Brand Identity

### Colors

**Primary Palette:**
```
Primary Blue: #0ea5e9 (rgb(14, 165, 233))
Accent Purple: #a855f7 (rgb(168, 85, 247))
Background Black: #000000
Text White: #ffffff
```

**Secondary Palette:**
```
Dark Blue: #0c4a6e
Light Blue: #38bdf8
Dark Purple: #7e22ce
Light Purple: #c084fc
```

**Utility Colors:**
```
Success Green: #10b981
Warning Yellow: #f59e0b
Error Red: #ef4444
Info Blue: #3b82f6
```

### Typography

**Primary Font**: Geist Sans (or system-ui fallback)
- Headlines: Bold, 48-72px
- Body: Regular, 16-18px
- Captions: Medium, 12-14px

**Monospace Font**: Geist Mono
- Code: Regular, 14px
- Addresses: Regular, 12px

### Logo Usage

**Primary Logo:**
- Square with "LB" text
- Gradient background (blue to purple)
- White text, bold, centered
- Minimum size: 32x32px
- Clear space: 8px all sides

**Logo Variations:**
- Light background: Use gradient version
- Dark background: Use gradient version (same)
- Small sizes: Use solid primary blue background

**Don'ts:**
- âŒ Don't rotate the logo
- âŒ Don't change colors
- âŒ Don't add effects (shadow, glow, etc.)
- âŒ Don't place on busy backgrounds

## Canva Templates

### Creating on Canva (Free)

1. **Sign up** at canva.com
2. **Create custom size**: 1200x675px
3. **Set background**: Black to #0c4a6e gradient
4. **Add elements**:
   - Logo (upload)
   - Text blocks
   - Icons (from Canva library)
5. **Save as template**
6. **Export as PNG** (high quality)

**Backgrounds**:
- Search: "dark gradient tech"
- Search: "minimalist black background"
- Search: "futuristic background dark"

**Icons**:

- Search: "ai icons minimalist"

**Fonts (if Geist not available)**:

- IBM Plex Sans
- Space Grotesk
- JetBrains Mono (for code)

## Video Guidelines (Future)

### Short Form (30-60s)

**Format**: 1080x1920px (vertical) or 1920x1080px (horizontal)

**Structure**:
1. Hook (3s): Attention grabber
2. Content (40s): Main message
3. CTA (7s): What to do next

**Style**:
- Fast-paced editing
- Subtitles (always)
- Dark theme
- Minimal text overlay
- Clear audio

### Long Form (5-10min)

**Format**: 1920x1080px (horizontal)

**Structure**:
1. Intro (30s): What's covered
2. Content (8min): Deep dive
3. Summary (1min): Key takeaways
4. CTA (30s): Next steps

**Style**:
- Professional production
- Screen recordings if tutorial
- Talking head if interview
- B-roll for interest
- Lower thirds for names
